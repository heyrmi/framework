PUT to https://httpbin.org/put

Body

{
    "status": "completed",
    "step": "3",
    "token": "abc123"
}

Headers

Accept: application/json, application/javascript, text/javascript, text/json
Content-Type: application/json

Curl

curl -v -X PUT 'https://httpbin.org/put' -H 'Accept: application/json, application/javascript, text/javascript, text/json' -H 'Content-Type: application/json' -d '{ "status": "completed", "step": "3", "token": "abc123" }'
