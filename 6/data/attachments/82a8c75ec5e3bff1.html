PATCH to https://httpbin.org/patch

Body

{
    "field": "patched-value"
}

Headers

Accept: application/json, application/javascript, text/javascript, text/json
Content-Type: application/json

Curl

curl -v -X PATCH 'https://httpbin.org/patch' -H 'Accept: application/json, application/javascript, text/javascript, text/json' -H 'Content-Type: application/json' -d '{ "field": "patched-value" }'
